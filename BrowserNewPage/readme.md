# New Page
Ein kleines Projekt für WMC  
  
## Overview
Dieses Projekt dient zum ersetzen der New Page, mit neuen features  
  
## Installation  
  
### Für Chromium basierte Browser:
chrome://extensions in der adressleiste ->  
-> Entwicklermodus aktivieren ->   
-> Entpackte Erweiterungen Laden ->   
-> Directory auswählen  
### Für Firefox Browser:  
about:preferences in der adressleiste ->  
-> Unter Startseite ->  
-> "Firefox-Startseite (Standard)" auf "Benutzerdefinierte URLs..." ->  
-> Directory Auswählen
